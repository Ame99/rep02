BDShop是国内少有前后端完全分离的java商城项目.
## BDShop 系统简介
BDShop商城系统是B2C模式的电子商城
后端基于springboot
前端基于nodejs
实现的前后端分离架构的项目

##BDShop技术路线
后端构建:spring-boot+mybatis+pagehelper+Swagger2构建RESTful API

前端构建:nodejs+gulp+requirejs+art-template+bootstrap+weui
